# Default Indian Host Images

This directory contains images for popular Indian podcast hosts. The following images should be added manually:

## Required Images:
- `ranveer-allahbadia.jpg` - Image for Ranveer Allahbadia (BeerBiceps)
- `nikhil-kamath.jpg` - Image for Nikhil Kamath (WTF with Nikhil Kamath)
- `raj-shamani.jpg` - Image for Raj Shamani (Figuring Out)
- `varun-mayya.jpg` - Image for Varun Mayya (Varun Mayya Podcast)
- `tanmay-bhat.jpg` - Image for Tanmay Bhat (Tanmay Bhat)
- `ankur-warikoo.jpg` - Image for Ankur Warikoo
- `preethi-srinivasan.jpg` - Image for Preethi Srinivasan (The Pragati Podcast)
- `shantanu-deshpande.jpg` - Image for Shantanu Deshpande (Bombay Journey)
- `janice-sequeira.jpg` - Image for Janice Sequeira (Social Podcast)
- `kushal-mehra.jpg` - Image for Kushal Mehra (The Seen and the Unseen)

## Image Guidelines:
- Format: JPG or PNG
- Recommended size: 300x300px or higher (square aspect ratio preferred)
- File size: Keep under 200KB for optimal loading

## Adding New Default Hosts:
1. Add the image file to this directory
2. Update the `DEFAULT_HOST_IMAGES` mapping in `PopularHosts.jsx`
3. Use the exact host name as the key

## Note:
If a host's image fails to load or doesn't exist, a placeholder avatar will be shown instead.
